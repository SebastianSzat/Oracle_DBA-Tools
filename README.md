# Oracle DBA Tools

A collection of Oracle DBA utilities for database administration and maintenance tasks.

## Projects

| Folder | Description |
|--------|-------------|
| [TNS_REPLACER](./TNS_REPLACER/) | Bulk find-and-replace tool for TNS entries across multiple Oracle database servers |
| [alertlog-viewer](./alertlog-viewer/) | Time-windowed Oracle alert log viewer with colour-highlighted terminal output |
| [ora-user-ensure](./ora-user-ensure/) | Verify and optionally create an Oracle user across a fleet of CDB/PDB/RAC databases |
| [data-generator](./data-generator/) | PL/SQL package that generates realistic test data into any Oracle schema using configurable column fill expressions |
| [data-anonymizer](./data-anonymizer/) | PL/SQL package that anonymizes sensitive data in staging databases while preserving referential integrity across parent-child tables |
